# Better Auth Template with Drizzle ORM

Local SQLite DB is used for this template.
To change.

## Requirements

- Node.js 20+
- npm
- Next
- Better-auth
- Drizzle ORM
- SQLite

## How to use

1. Clone the repo
2. Run `npm install`
3. Run `npm run dev`
4. copy .env.example to .env and change the variables
5. Open [http://localhost:3000](http://localhost:3000)
